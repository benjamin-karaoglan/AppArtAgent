# Getting Started

This section covers everything you need to get Appartment Agent running on your machine.

## Overview

Appartment Agent can be run in two ways:

1. **Docker (Recommended)**: Full stack with all services pre-configured
2. **Local Development**: Individual services for development and debugging

## Sections

| Guide | Description |
|-------|-------------|
| [Quick Start](quickstart.md) | Get running in 5 minutes with Docker |
| [Prerequisites](prerequisites.md) | Required tools and accounts |
| [Configuration](configuration.md) | Environment variables and settings |

## Choose Your Path

### I want to try it out quickly

Follow the [Quick Start](quickstart.md) guide. You'll have everything running in about 5 minutes.

### I want to develop or contribute

1. Review [Prerequisites](prerequisites.md) for development tools
2. Follow [Local Setup](../development/local-setup.md) for detailed instructions
3. Enable [Hot Reload](../development/hot-reload.md) for efficient development

### I want to deploy to production

1. Complete [Quick Start](quickstart.md) to verify everything works locally
2. Follow [GCP Deployment](../deployment/gcp.md) for production deployment
