## Getting Started

1. **Fork** the repository
2. **Clone** your fork locally
3. **Set up** the development environment (see [Development Guide](./docs/development/local-setup.md))
4. **Create a branch** for your changes
5. **Make changes** and test them
6. **Submit a Pull Request**

## Development Setup

```bash
# Clone your fork
git clone https://github.com/benjamin-karaoglan/appartment-agent.git
cd appartment-agent

# Start development environment
docker-compose up -d

# Or see docs/development/local-setup.md for local setup
```

## Pull Request Process

1. Update documentation if needed
2. Add tests for new functionality
3. Ensure all tests pass
4. Update the CHANGELOG if applicable
5. Request review from maintainers

## Code Style

- **Python**: Follow PEP 8, use Black formatter
- **TypeScript**: Use ESLint configuration
- **Commits**: Use conventional commit messages
